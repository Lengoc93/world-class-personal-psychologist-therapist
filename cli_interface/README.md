## Available Commands

| Command | Purpose | Time |
|---------|---------|------|
| `/therapy-session` | Full therapy conversation | 20-45 min |
| `/daily-check` | Quick wellness check | 5 min |
| `/create-roadmap` | Build 3-6 month development plan | 30 min |
| `/view-roadmap` | Review progress & adjust goals | 10 min |
| `/update-profile` | Update psychological profile | 10 min |
| `/load-diary-context` | One-time historical diary load | 15-30 min |

## Quick Start

1. **Fill your profile**: Edit `profile/personality_assessment.json`
2. **Start first session**: Type `/therapy-session`
3. **Create roadmap**: Type `/create-roadmap`
4. **Daily use**: Use `/therapy-session` or `/daily-check`

## Why Use CLI Interface?

✅ **Integrated workflow** - Works within Claude Code CLI
✅ **Slash commands** - Quick access to all features
✅ **Automated saving** - Sessions auto-save to local files
✅ **Terminal-based** - Perfect for developers and CLI users
✅ **Scriptable** - Can be integrated into workflows
