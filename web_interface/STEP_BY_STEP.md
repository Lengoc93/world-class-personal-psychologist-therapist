## Detailed Version with Screenshots

### Step 1: Open Claude.ai
```
🌐 Browser → https://claude.ai
👤 Sign in with your account
```

### Step 2: Start New Conversation
```
Click: [+ New conversation] button
(Usually top left or top right)
```

### Step 3: Attach Your Profile Files
```
Look for: 📎 (paperclip icon)
Click it

Navigate to your project folder:
📁 world class personal psychologist therapist
  └─ 📁 profile
      ├─ ✅ personality_assessment.json ← SELECT THIS
      └─ ✅ preferences.json ← SELECT THIS

Click "Open" or "Attach"
```

**What you'll see:**
```
[📎 personality_assessment.json]
[📎 preferences.json]
```

### Step 4: Get the Therapy Prompt
```
On your computer:
📁 world class personal psychologist therapist
  └─ 📁 web_interface
      └─ 📄 THERAPY_SESSION_PROMPT.md

Open this file in any text editor
```

### Step 5: Copy the Prompt
```
In THERAPY_SESSION_PROMPT.md:
1. Find the section: "## 📋 PROMPT TO COPY"
2. Select everything below that
3. Copy (Ctrl+C / Cmd+C)
```

The prompt starts with:
```
You are Dr. Sarah Chen, a world-class psychologist...
```

And ends with:
```
**Begin the daily check-in now.**
```

### Step 6: Paste in Claude
```
Back to claude.ai:
1. Click in the message box
2. Paste (Ctrl+V / Cmd+V)
3. Press Enter
```

### Step 7: Talk with Dr. Chen!
```
Dr. Chen will greet you.

Then you can reply in:
✅ Tiếng Việt: "Chào chị, hôm nay em cảm thấy..."
✅ English: "Hi Dr. Chen, today I feel..."
✅ Vietish: "Hi chị, hôm nay em feel..."

ANY language works!
```

## Example First Conversation

### What You'll See:

**You paste the prompt and press Enter**

**Dr. Chen responds:**
```
Hello, I'm Dr. Sarah Chen. It's wonderful to meet you.
I've read your profile and I'm here to support you.

Chào em, mình là Dr. Sarah Chen. Rất vui được gặp em.
Mình đã đọc profile của em và sẵn sàng hỗ trợ em.

Let's start with a few questions:

1. On a scale of 1-10, how are you feeling today?
   (Từ 1-10, hôm nay em cảm thấy thế nào?)

2. What's your energy level like?
   (Năng lượng của em như thế nào?)

3. How did you sleep last night?
   (Đêm qua em ngủ thế nào?)
```

**You can respond:**
```
Mood: 6/10
Energy: 5/10
Sleep: 7/10

Hôm nay em feel okay but có chút overwhelmed vì work.
```

**Dr. Chen continues:**
```
I hear you - feeling okay overall but some overwhelm around work.
That 6/10 mood with moderate energy makes sense given what you're
experiencing.

Tell me more about what's feeling overwhelming at work?
```

**And the conversation flows naturally from there!**

## Troubleshooting

### ❌ Can't find paperclip icon?
- Look for "Attach files" or upload button
- On mobile: Try "Request Desktop Site" in browser menu

### ❌ Files won't upload?
- Make sure files are .json format
- Try refreshing the page
- Use desktop browser if on mobile

### ❌ Dr. Chen responds in wrong language?
- Just say: "Please respond in Vietnamese" or "Vui lòng trả lời bằng Tiếng Việt"
- She'll adjust immediately

### ❌ Dr. Chen doesn't seem to know me?
- Check that you attached personality_assessment.json
- This file is crucial!

### ❌ Want to start over?
- Click "New conversation" anytime
- Attach files again
- Paste prompt again
- Fresh start!
